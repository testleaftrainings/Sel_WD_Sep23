    Steps to create a Maven Project
    1. File -> New -> Project -> Maven Project
    2. Click Next -> Click on Create a simple project checkbox
    3. Enter Group Id -> organiztion name
    4. Enter Artifact Id -> Project Name
    5. Click Finish

    Steps to create a Package
    Right click on src/main/java -> New -> Package

    Steps to create a Class
    Right click on Package -> New -> Class
